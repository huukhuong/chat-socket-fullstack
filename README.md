# CHAT APP FULL STACK

### Server: Ubuntu 22.04
<ul>
<li>NGINX</li>
<li>PM2</li>
<li>phpMyAdmin</li>
</ul>

### Back end: NestJS

#### Technologies:

<ul>
<li>Websocket</li>
<li>TypeORM</li>
<li>MySQL</li>
<li>JWT Token</li>
<li>Swagger</li>
</ul>

### Mobile: React Native

#### Technologies:

<ul>
<li>Socket.io</li>
<li>TailwindCSS (NativeWind)</li>
<li>Redux Toolkit</li>
<li>React Query</li>
<li>Reanimated v3</li>
<li>React Navigation</li>
<li>Flashlist</li>
</ul>

### Features

<ul>
<li>Login using JWT</li>
<li>Send/accept friend request</li>
<li>Get realtime online/offline status</li>
<li>Chat between 2 users (must be friends)</li>
<li>Send and receive image, text</li>
<li>Upload story like Facebook</li>
</ul>
